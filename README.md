## Опис роботи

У межах практичної роботи розроблено веб-калькулятор для:

- перерахунку елементарного складу палива  
  з робочої маси на суху та горючу;
- обчислення коефіцієнтів переходу;
- визначення нижчої теплоти згоряння.

Застосунок реалізовано мовою **Go** з використанням стандартної бібліотеки `net/http` та `html/template`.

---

## Реалізовані розрахунки

### Коефіцієнти переходу

Перехід від робочої маси до сухої:

Kрс = 100 / (100 − Wᵖ)

Перехід від робочої маси до горючої:

Kрг = 100 / (100 − Wᵖ − Aᵖ)

---

### Склад сухої маси

Hᶜ = Hᵖ · Kрс  
Cᶜ = Cᵖ · Kрс  
Sᶜ = Sᵖ · Kрс  
Nᶜ = Nᵖ · Kрс  
Oᶜ = Oᵖ · Kрс  
Aᶜ = Aᵖ · Kрс

---

### Склад горючої маси

Hᵍ = Hᵖ · Kрг  
Cᵍ = Cᵖ · Kрг  
Sᵍ = Sᵖ · Kрг  
Nᵍ = Nᵖ · Kрг  
Oᵍ = Oᵖ · Kрг

---

### Нижча теплота згоряння (формула Менделєєва)

Qᵣⁿ =  
(339·Cᵖ + 1030·Hᵖ − 108.8·(Oᵖ − Sᵖ) − 25·Wᵖ) / 1000

---

### Перерахунок теплоти

Для сухої маси:

Qˢⁿ = ((Qᵣⁿ + 0.025·Wᵖ) · 100) / (100 − Wᵖ)

Для горючої маси:

Qʰⁿ = ((Qᵣⁿ + 0.025·Wᵖ) · 100) / (100 − Wᵖ − Aᵖ)
